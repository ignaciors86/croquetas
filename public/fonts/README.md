# Fuentes

Coloca aquí los archivos de fuente para las croquetas y el prompt:

## Para textos de croquetas:
- `CroquetaFont.woff2`
- `CroquetaFont.woff` (opcional)
- `CroquetaFont.ttf` (opcional)

## Para el prompt:
- `PromptFont.woff2`
- `PromptFont.woff` (opcional)
- `PromptFont.ttf` (opcional)

Los font-faces ya están configurados en `app/globals.css`. Solo necesitas añadir los archivos de fuente aquí.
